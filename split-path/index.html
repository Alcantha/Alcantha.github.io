Path

[M] turn [newPathName] [color] [offset] [: conditions]
...

Start Path

START

(x, y) : pathName : [M] direction [color] [offset], ... [: startProps]
...

Values

  • color: colorName or cssColor
  • colorName: color pattern name
  • cssColor: valide CSS color (eg: black, #000)
  • pathName: path pattern name
  • newPathName: new path to change from the current path
  • turn:
    • above 0: number of clockwise turn by 45 degres
    • 0: straight line
    • below 0: number of anticlockwise turn by 45 degres
  • offset:
    • above 0: conditions are moved forward by n cycles
    • 0: no offset (default)
    • below 0: conditions are delayed by -n cycles, go in straight line during delay
  • direction:
    • E / SE / S / SW / W / NW / N / NE: starting directions for drawing
    • <int>: 0 starts at East and positive value turns clockwise by 45 degrees

Parameters

conditions
(after | before | every | only | start) = n, ...
  • after: valid after cycle n
  • before: valid before cycle n
  • every*: valid every n cycle
  • only*: valid at specific n cycle
  • start: valid starting at n cycle, can be used to shift every
*: may contain multiple values separated by space
pathProps
(color | max) = n, ...
  • color: see color
  • max: draw until cycle n
startProps
(color | max | offset | startpoint) = n, ...
  • color: see color
  • max: draw until cycle n
  • offset: jump by n cycle forward, value below 0 will be in straight line
  • startpoint: exclude the start point when set to 0

Start position

x
a | a + b | a - b | a * b | a / b | a % b
Values for a / b:
  • <int>: positive value is directed to the right
  • LEFT/L: left most value of the grid
  • RIGHT/R: right most value of the grid
y
a | a + b | a - b | a * b | a / b | a % b
Values for a / b:
  • <int>: positive value is directed to the bottom
  • UP/U: up most value of the grid
  • BOTTOM/B: bottom most value of the grid
